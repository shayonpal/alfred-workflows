# Open Emoji Keyboard

Just a simple addition of an extra key combination to open macOS’ emoji keyboard.

**macOS shortcut:** ⌘ + ^ + Space

**Shortcut this workflow adds:** ⌘ + ⌥ + E (customisable within the workflow)
